# Unity 2D Roguelike Prototype

This repository contains a **beginner-level 2D roguelike game prototype** developed with **Unity and C#** as a **small group learning project**.

The project was **initially based on a guided tutorial**, which our team intentionally chose to learn game development from an experienced source instead of attempting to design all systems from scratch.  
The goal was to understand how real-world Unity projects are structured and how gameplay systems interact.

After following the tutorial, the team **experimented with small changes, adjustments, and extensions** to better understand the mechanics and code flow.
